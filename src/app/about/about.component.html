LARO was created as my personal project for the Front-End Web Development track of LaunchCode's CoderGirl program.

It is a board game randomizer that uses Board Game Atlas' API that contains an array of 100 board games.

It uses JavaScript / TypeScript with an Angular framework and a Bootstrap v3.4.1 theme from Bootswatch.
